# Scalentia Bot — инструкция запуска

## 1. Получи токен бота
1. Напиши @BotFather в Telegram
2. /newbot → введи имя → введи username (например scalentia_bot)
3. Скопируй токен вида `7123456789:AAF...`

## 2. Узнай свой Telegram ID
Напиши @userinfobot — он пришлёт твой числовой ID

## 3. Вставь данные в bot.py
Открой bot.py, найди строки:
```
BOT_TOKEN = "ВСТАВЬ_ТОКЕН_БОТА"
OWNER_ID  = 123456789
OWNER_NICK = "@scalentia"
```
Замени на свои данные.

## 4. Вставь реквизиты оплаты
Найди строку:
```
<code>Тинькофф: +7 XXX XXX XX XX</code>
```
Замени на свой номер карты/телефона.

## 5. Запуск на сервере (Railway — бесплатно)

### Вариант A: Railway.app (рекомендую)
1. Зайди на railway.app
2. New Project → Deploy from GitHub
3. Загрузи эти файлы на GitHub
4. Railway сам установит зависимости и запустит

### Вариант B: Локально (для теста)
```bash
pip install -r requirements.txt
python bot.py
```

## Что бот умеет
- Показывает 4 шаблона ботов с ценами и описанием
- Принимает заявки на покупку → уведомляет тебя
- Ведёт воронку для индивидуального бота (ниша → бюджет → задача)
- Пересылает тебе вопросы клиентов
- При ошибке — уведомляет тебя и пишет клиенту

## Как добавить свои шаблоны
В файле bot.py найди TEMPLATES = { ... }
Добавь новый блок по аналогии.
