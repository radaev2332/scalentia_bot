## Запуск за 15 минут

### 1. Получи токен
@BotFather → /newbot → скопируй токен

### 2. Узнай свой ID  
Напиши @userinfobot — скопируй числовой ID

### 3. Вставь в файл (первые 3 строки)
```python
BOT_TOKEN  = "7123456789:AAF..."
OWNER_ID   = 387654321
OWNER_NICK = "@scalentia"
```

### 4. Настрой данные под клиента
Каждый бот — имена, услуги, цены, адрес — всё в переменных в начале файла

### 5. Запуск (бесплатный хостинг)
railway.app → New Project → залей файл → готово, работает 24/7

## Для локального теста
```
pip install python-telegram-bot==20.7
python barber_bot.py
```

## Что получает владелец бизнеса
Каждый бот при любом действии клиента (запись, покупка, вопрос) — 
присылает тебе уведомление с именем и ником клиента в Telegram.
